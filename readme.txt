A plugin to provide a simple statistics page for Mailpoet 3 Subscribers. 

== Description ==

This plugin simply adds in a page to view the statistics associated with a single subscriber. From the MP Subscriber Statistics menu item, you can view a chart.js graph of their stats, and filter a list of all the newsletters sent to them.

== Installation ==
= Minimum requirements =
PHP version 5.6 or greater. 

There are 3 ways to install this plugin:

= 1. Install Through Wordpress =
1. In your WordPress dashboard, navigate to Plugins > Add New
1. Search for `Mailpoet Subscriber Overview`
1. Click on "install now"
1. Activate the plugin
1. A new `MP Subscriber Statistics` menu will appear in your WordPress dashboard

= 2. Zip File =
1. Download the plugin (.zip file) by using the blue "download" button underneath the plugin banner at the top
1. In your WordPress dashboard, navigate to Plugins > Add New
1. Click on "Upload Plugin"
1. Upload the .zip file
1. Activate the plugin
1. A new `MP Subscriber Statistic` menu will appear in your WordPress dashboard

= 3. FTP =
1. Download the plugin (.zip file) by using the blue "download" button underneath the plugin banner at the top
1. Extract the archive and then upload, via FTP, the `mailpoet` folder to the `<WP install folder>/wp-content/plugins/` folder on your host
1. Activate the plugin
1. A new `MP Subscriber Statistic` menu will appear in your WordPress dashboard
